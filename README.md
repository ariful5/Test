# 🚀 JeevanSathi OTP অটোমেশন টুল

> দ্রুত এবং সহজ OTP অটোমেশন

## 📥 ইনস্টলেশন

```bash
git clone https://github.com/YOUR_USERNAME/jeevansathi-otp-automation.git
cd jeevansathi-otp-automation
npm install
```

## 🎯 ব্যবহার

### একক নম্বরে OTP পাঠান:
```bash
npm start
```

### কোড এডিট করুন (`app.js`):
```javascript
const phoneNumbers = [
    '9102808298',  // আপনার নম্বর এখানে লিখুন
];
```

## 📋 ফিচার

✅ স্বয়ংক্রিয় OTP পাঠানো  
✅ প্রোফাইল ভেরিফিকেশন  
✅ ব্যাচ প্রসেসিং  
✅ এরর হ্যান্ডলিং  

## ⚙️ কনফিগারেশন

`app.js` এ পরিবর্তন করুন:
- `phoneNumbers` - নম্বরের লিস্ট
- `isd` - দেশ কোড (ডিফল্ট: ৯১)
